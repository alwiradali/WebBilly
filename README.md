# Billy Digitals

Premium web development agency website — static site (HTML/CSS/JS), no build step.

## Run locally

Open `index.html` in a browser, or serve the folder:

```
python -m http.server 4173
```

## Configure contact details

Edit `CONFIG` at the top of `assets/js/main.js`:

- `email` — enquiry email address
- `whatsapp` — WhatsApp number (country code + number, digits only, e.g. `447700900123`)

## Features

- AI chat assistant ("Billy") — answers questions about services, plans, timelines
- Email + WhatsApp enquiry channels (form pre-fills an email)
- Standard / Premium / Ultra Premium plans (no prices shown — quote on request)
- Sample template styles for 10 industries with category filtering
- Fully responsive, dark premium theme, scroll animations, FAQ, testimonials
