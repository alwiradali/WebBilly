# Billy Digitals — Logo Kit

**Founder:** Bilawal Ali · **Site:** billydigitals.com

## The mark
A "B" built from a **browser window** (top) + an **app/device card** (bottom) — web + mobile, the two things Billy Digitals builds.

## Files
| File | Use |
|------|-----|
| `logo-full-onlight.png` | Full logo for **light** backgrounds (dark wordmark) |
| `logo-full-ondark.png`  | Full logo for **dark** backgrounds (white wordmark) |
| `logo-icon.png`         | Icon only, transparent (favicons, watermarks) |
| `logo-badge.png`        | App icon / social profile picture (rounded dark tile) |
| `logo-icon.svg`         | Scalable vector icon (perfect at any size) |

All PNGs are transparent and rendered at 4× for print & retina.

## Brand colours
- Gradient: **#2B7FFF → #22BFF0 → #22D3EE** (blue → cyan), 125°
- Ink (dark text): **#0B1020**
- Wordmark: bold geometric sans, tight tracking

## Don'ts
- Don't stretch or recolour the mark. Keep clear space of ~½ the icon height around it.
