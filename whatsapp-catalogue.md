# Billy Digitals — WhatsApp Business Catalogue

Ready-to-paste product list for the WhatsApp Business app. Each product has a
matching branded image in `assets/catalogue/`.

## How to add these in the app
**WhatsApp Business → Settings → Business tools → Catalogue → `+ Add item`**, then
for each product below: add the **image**, **name**, **price**, **description** and
**link** → **Save**.

Notes:
- Set the catalogue **currency to GBP (£)**.
- Web-design plans are **quote-based** — leave the price field blank (or set to 0
  and hide it) so each item reads "Free quote". Hosting and care are **per month**
  with fixed prices.
